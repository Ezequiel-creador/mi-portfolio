Crud

Este proyecto es una aplicación web completa que permite la gestión del registro de usuarios mediante operaciones CRUD (Crear, Leer, Actualizar, Eliminar). El frontend está desarrollado utilizando HTML, CSS y AJAX para ofrecer una interfaz de usuario interactiva y dinámica. El backend está construido con Spring Boot, y la base de datos es gestionada mediante MySQL Workbench.
